# Onboarding A New Client

Use this page as the starting checklist for a new BAS Platform installation.

## Checklist

1. Confirm business scope and required modules.
2. Collect source systems and data samples.
3. Identify product identity fields.
4. Define catalog, pricing, and export requirements.
5. Define user roles and module access.
6. Configure imports and run dry-runs.
7. Configure channels and exports.
8. Prepare support handover notes.
