# Deployment Model

The recommended primary model is managed single-tenant SaaS.

## Managed Single-Tenant SaaS

- one codebase;
- one installation per customer;
- separate database per customer;
- operated on BAS Platform infrastructure;
- customer does not access server, Docker, or source code.

## Enterprise On-Premise

For large customers, on-premise deployment can be offered with stricter operational and licensing rules.
