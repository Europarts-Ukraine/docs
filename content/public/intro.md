# PAD Platform Documentation

Welcome to the PAD Platform documentation.

PAD Platform is a product data automation platform for ecommerce teams. It turns data from ERP systems, CSV files, supplier feeds, APIs, and databases into governed, repeatable product-data workflows.

This public documentation is written for system users and customer-side administrators. It explains how to work in the platform, how to understand statuses and errors, and where to look when something does not behave as expected.

## Documentation Areas

- **Customer Help Center** explains day-to-day workflows in the PAD Platform interface.
- **Integrations** explains external API and export contracts when a customer or partner integrates with the platform.

Internal engineering, implementation, deployment, and source-code documentation is not part of the public documentation.
