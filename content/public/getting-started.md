# Getting Started

This section introduces the main areas of BAS Platform.

## Main Workflow

```text
Data sources
  -> imports and validation
  -> product catalog / PIM
  -> mapping and identity resolution
  -> pricing
  -> translations and enrichment
  -> channels
  -> exports
```

## Typical Users

- catalog managers;
- content managers;
- pricing managers;
- import and integration specialists;
- ecommerce administrators;
- support users who investigate issues.

## First Things To Learn

1. How product data is organized in the catalog.
2. How imports are checked before they update business data.
3. How mapping groups related product records.
4. How price types and pricing runs work.
5. How channels and exports decide what data is published.
