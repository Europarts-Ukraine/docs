# Troubleshooting

This page collects common user-facing problems and first checks.

## I Do Not See A Module

Check your role and module permissions. Ask an administrator to verify your access.

## Import Failed

Open the import run and review blocking issues. Fix the source data or import profile, then run a dry-run before applying changes.

## Prices Did Not Update

Check the price calculation run, calculation errors, active rules, base prices, and target price type.

## Export Is Empty

Check profile diagnostics, channel scope, channel readiness, required prices, and translations.

## Product Data Looks Outdated

Check whether the latest source import finished successfully and whether a newer source value was blocked by validation or manual approval.
